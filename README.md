# Particle Portfolio

マウスの動きに反応するパーティクルが浮遊し、近接パーティクル同士が線でつながるインタラクティブなポートフォリオサイト。

## 使用技術

- HTML5 (セマンティック構造)
- CSS3 (CSS変数、Grid、Flexbox、アニメーション)
- JavaScript (Canvas 2D API、IntersectionObserver)
- 外部ライブラリ不使用

## 機能一覧

- **パーティクルシステム**: Canvas 2D で描画されるパーティクルが画面上を浮遊
- **マウスインタラクション**: マウス周辺のパーティクルが反発し、動的に反応
- **接続線描画**: 近接パーティクル間を線で接続（距離に応じて透明度が変化）
- **レスポンシブ対応**: モバイル〜デスクトップまで最適化（パーティクル数も自動調整）
- **スクロールアニメーション**: IntersectionObserver によるフェードイン表示
- **モバイルナビゲーション**: ハンバーガーメニューによるナビ切り替え
- **タッチ対応**: モバイルでもタッチ操作でパーティクルが反応

## ファイル構成

```
particle-portfolio/
├── index.html      # メインHTML
├── css/
│   └── style.css   # スタイルシート
├── js/
│   └── main.js     # パーティクルシステム＋インタラクション
└── README.md       # このファイル
```

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要。

```bash
open index.html
```
